= SnapCenter REST API にネイティブでアクセスする方法
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter REST API には、 REST クライアントをサポートする任意のプログラミング言語を使用して直接アクセスできます。代表的な言語の選択肢は、 Python 、 PowerShell 、 Java です。
