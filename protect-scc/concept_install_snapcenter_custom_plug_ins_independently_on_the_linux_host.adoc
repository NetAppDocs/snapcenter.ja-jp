= SnapCenter Custom Plug-ins を Linux ホストに個別にインストールします
:icons: font
:imagesdir: ../media/


[role="lead"]
環境によっては、 SnapCenter サーバから SnapCenter Custom Plug-ins をリモートでインストールできない場合は、 Linux ホストに直接インストールできます。対話型のウィザードまたはコマンドラインインターフェイスを使用してプラグインをインストールできます。
