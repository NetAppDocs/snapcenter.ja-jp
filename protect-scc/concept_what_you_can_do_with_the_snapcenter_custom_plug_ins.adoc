= SnapCenter Custom Plug-ins の機能
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter Custom Plug-ins を使用してデータ保護処理を実行できます。

* データベース、インスタンス、ドキュメント、表領域などのリソースを追加します。
* バックアップを作成します
* バックアップからリストアします
* バックアップをクローニングする。
* バックアップ処理のスケジュールを設定します。
* バックアップ、リストア、クローニングの各処理を監視する。
* バックアップ、リストア、クローニングの各処理のレポートを表示します。
