= インストルチユウニLinuxホストテユウコウニナルキノウ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenterサーバは以下のソフトウェアパッケージをインストールします。これらのパッケージは、トラブルシューティングやホストシステムのメンテナンスに役立ちます。

* RabbitMQ
* nginx
* アーラン
* .NET Runtime 8.0.12（およびそれ以降のすべての8.0.xパッチ）
* PAM -デベル
* PowerShell
