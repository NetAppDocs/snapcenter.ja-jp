= インストルチユウニLinuxホストテユウコウニナルキノウ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenterサーバは以下のソフトウェアパッケージをインストールします。これらのパッケージは、トラブルシューティングやホストシステムのメンテナンスに役立ちます。

* RabbitMQ
* nginx
* アーラン
* .NET Frameworkバージョン8.0.5以降、以降のすべての.NET 8パッチを含む
* PAM -デベル
* PowerShell
