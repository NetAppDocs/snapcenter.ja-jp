= SnapCenterによる不要なデータバックアップの削除の管理方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenterは、ストレージシステムレベルおよびファイルシステムレベルでの不要なデータバックアップの削除を管理します。

保持設定に基づいて、プライマリストレージまたはセカンダリストレージ上のSnapshotと、IBM DB2カタログ内の対応するエントリが削除されます。
