= 自動検出されたIBM DB2でサポートされるリストア戦略のタイプ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenterを使用してリストア処理を正常に実行するには、戦略を定義する必要があります。

完全リソースリストアは、自動的に検出されたIBM DB2データベースに対してサポートされるリストア戦略です。これにより、リソースのすべてのボリューム、qtree、およびLUNがリストアされます。
